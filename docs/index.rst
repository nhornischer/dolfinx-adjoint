DOLFINx-ADJOINT Documentation
=============================

An automatic differentiation library for DOLFINx that computes gradients using
the adjoint method.

Overview
--------

This site contains:

- Project overview and installation guidance
- API documentation generated from ``src/dolfinx_adjoint``
- Documentation CI and publishing workflow references

Quick Start
-----------

.. code-block:: bash

   git clone https://github.com/nhornischer/dolfinx-adjoint.git
   cd dolfinx-adjoint
   pip install -e .
